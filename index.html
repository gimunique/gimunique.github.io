ABOUT

안녕하세요. 디자인에 디테일을 입혀 정교한 인터페이스와 애니메이션으로 화면을 완성하는 UI 개발자 김유진입니다.
대학에서 순수 미술을 전공하고 디자이너 길을 거쳐 정적인 디자인이 코딩을 통해 동적으로 화면단에서 반응하는 것에 매료되어 웹퍼블리셔의 길로 접어들게 되었습니다.
빠르게 변하는 웹트랜드에 맞춰 끊임없이 벤치마킹을 통해 새로운 변화를 시도하고 있으며, 주도적으로 업무를 진행할 수 있는 환경을 선호합니다.
또한 초심을 잃지 않고 늘 노력하는 자세로 프로의식과 책임감을 가지고 작업에 임하고 있습니다.

  • 01
    HTML / CSS
    • 검색엔진에 최적화(SEO)와 접근성이 높은 시멘틱 마크업이 가능합니다.
    • CSS3를 이용한 인터렉티브한 페이지 코딩이 가능합니다.
    • 다양한 플랫폼 및 브라우저에 최적화된 웹표준, 웹호환(크로스브라우징) 준수가 가능합니다.
  • 02
    JAVASCRIPT
    • ES5 이후의 자바스크립트 문법에 익숙합니다.
    • jQuery 라이브러리를 통한 이벤트 처리에 능숙합니다.
    • javascript 라이브러리 사용에 능숙합니다.
    • 유저 인터페이스를 고려한 센스있는 애니메이션 구현이 가능합니다.
    • OPEN API 사용이 가능합니다.
  • 03
    Reponsive Web
    • 디스플레이에 최적화된 페이지 코딩이 가능합니다.
    • 스크립트를 사용하지 않고 순수 css만으로 반응형 웹 코딩에 능숙합니다.
  • 04
    Design
    • UI, UX에 대한 이해도가 높습니다.
    • 디자인 툴(Photoshop) 사용에 익숙합니다.
  • 05
    Etc
    • SVN을 통한 프로젝트 버전 관리가 가능합니다.
    • 최신 트렌드 숙지를 위한 벤치마킹을 게을리하지 않습니다.
    • 업무에 필요한 부분이 있다면 능숙한 분야가 아니더라도 적극적인 자세로 최적의 결과를 낼 수 있도록 노력합니다.
